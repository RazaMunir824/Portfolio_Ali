Projects

Face Recognition Brain

This project uses the Clarifai API to detect a face in a photo. It then draws a bounding box around the face and also adds a count for the number of time a user scans a photo. This is then stored in a PostgreSQL database along with the user Information. It also uses hashes to save the password to increase security.

  • Styling was done using tachyons
  • NPM was used with create-react-app to create the initial project structure
  • App was hosted using Heroku
  • Uses PostgreSQL as the database
